Εγκατάσταση ebib

Ένας τρόπος είναι το Ebib: a BibTeX database manager for Emacs 1. Αυτό (όπως συμβαίνει τις περισσότερες φορές) με ογήφησε σε άλλους δρόμους και ανακάλυψα το Melpa [fb:melpa] μέσω του οποίου μπορείς να εγκαταστείς πολύ ευκολα πακέτα στο Emacs (M-x package-list-packages).

Στις [2014-02-09 Sun] υπήρχε πρόβληλα στην εγκατάσταση και δημιούργησα ένα καινούργιο issue στο https://github.com/joostkremers/ebib/issues/41.

Λεξικό - Manual

Έτσι έχοντας πλέον εγκαταστήσει το ebib, μπορούμε να το τρέξουμε. Παρακάτω θα δημιουργήσουμε ένα λεξικό με τις εντολές και τις συντομέυσεις αυτών για γρήγορη περιήγηση στο ebib.

M-x ebib start ebib  
q quit  
z leave  
o loading .ebib file  
? help  
  insert key ebib-insert-insert-bibtex-key
C-s <RET> <RET> search  

Πως εισάγουμε βιβλιογραφική αναφορά;

C-c [

Το συντακτικό που χρησιμοποιούμε για την εισαγωγή μια βιβλιογραφικής αναφοράς στο orgmode είναι το ακόλουθο: 

[[ebib:roads_microsound_2004][Microsound cite]]

Zotero

Σημαντικό στοιχείο αποτελεί το αρχείο .bib που στη περίπτωση μας εξάγεται αυτόματα απο το zotero 2. Τα Ελληνικά γενικά είναι πρόβλημα μιας και οι υπολογιστές μιλάνε Αγγλικά. Το όνομα με το οποίο αποθηκεύεται το βιβλίο-δημοσίευση κ.α. είναι συγγραφέας_πρώτη λέξη τίτλου_χρονιά.

Επίσης οι σημειώσεις μέσα στο zotero πρέπει να έχουν συγκςκριμένη μορφή για να μπορούν να αναγνωριστούν απο το ebib.

Όλες οι βιβλιογραφικές καταχωρήσεις συλλέγονται σε ένα αρχείο και κατατάσονται βάση των tags.
